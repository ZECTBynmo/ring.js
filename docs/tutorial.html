Installation in Browser

Download the last release of Ring.js. Then extract the zip file.

Ring.js has only one dependency: Underscore.js. The underscore library is already provided in the Ring.js release but you can use another version if you want.

To use Ring.js add theses lines to your HTML header:

<script type="text/javascript" src="underscore.js"></script>
<script type="text/javascript" src="ring.js"></script>

Ring.js is also compatible with Require.js. Simply load the ring.js file and it will call the define() method instead of defining a global variable.

Declaring Classes

The ring.class() function is used to declare new classes:

var A = ring.class({
    saySomething: function() {
        return "hello world";
    },
});
var a = new A();
console.log(a.saySomething());
// will print "hello world" in the JavaScript console

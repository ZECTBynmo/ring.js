Then extract the zip file. Ring.js has only one dependency: [Underscore.js](http://underscorejs.org/). The underscore library is already provided in the Ring.js release but you can use another version if you want. To use Ring.js add theses lines to your HTML header: :::html Ring.js is also compatible with [Require.js](http://requirejs.org/). Simply load the `ring.js` file and it will call the `define()` method instead of defining a global variable.
